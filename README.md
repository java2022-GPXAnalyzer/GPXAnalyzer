## How to develop

### Frontend

Version: Windows, node.js >= v16.15.0, npm >= v8.5.5 (LTS version) \\
```bash
git clone https://github.com/java2022-GPXAnalyzer/GPXAnalyzer.git
cd frontend
yarn install && yarn cesium-fix
yarn electron:serve
```

### Backend

Version: Windows, Java SE 17 \\
```bash
git clone https://github.com/java2022-GPXAnalyzer/GPXAnalyzer.git
cd backend
mvnw spring-boot:run
```
